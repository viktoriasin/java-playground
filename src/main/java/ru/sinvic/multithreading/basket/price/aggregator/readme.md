# Агрегатор стоимости корзины интернет-сервиса

Вам нужно реализовать сервис расчёта стоимости корзины интернет‑магазина. В корзине может быть до 1 000 товаров, и для каждого нужно:

- получить актуальную цену из внешнего сервиса;

- проверить наличие промоакции (скидка, кешбэк, бонусы);

Далеe надо рассчитать итоговую стоимость корзины с учётом всех скидок.

Требования:

- Все товары должны обрабатываться параллельно.

- Время расчёта корзины — не более 2 сек независимо от количества товаров.

- Если какой‑то товар не удалось обработать (сервис недоступен), его стоимость должна быть 0, но остальные товары учитываются.

- Необходимо поддержать три типа промоакций, которые могут применяться одновременно:

    - фиксированная скидка (например, 100 руб);

    - процентная скидка (например, 20 %);

    - кешбэк (например, 5 % от суммы).

Итоговая сумма должна включать:

- сумму товаров с учётом скидок;

- общий кешбэк;

- налог (20 % от итоговой суммы без кешбэка).

Код должен быть тестируемым и не зависеть от реальных сетевых вызовов.
