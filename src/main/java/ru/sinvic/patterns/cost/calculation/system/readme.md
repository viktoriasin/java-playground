## Условие
Вам нужно реализовать модуль расчёта стоимости доставки для интернет‑магазина. 
Стоимость зависит от типа доставки и параметров заказа (вес, расстояние).

## Требования:

Поддерживать 3 типа доставки:

- Курьерская (Courier): фиксированная цена + надбавка за вес.

- Почта (Post): цена зависит от расстояния (тариф за км).

- Самовывоз (Pickup): бесплатно.

Для каждого типа доставки должна быть своя формула расчёта.

Система должна легко расширяться (добавление нового типа доставки без изменения существующего кода).

Клиентский код не должен знать детали реализации — только интерфейс расчёта.

Написать юнит‑тест, проверяющий все 3 стратегии.
